# Unbroken-Checkin-2
Periodic check-in assessment for UnBroken program participants. Built with HTML, deployed via Netlify, connected to Airtable for outcome tracking.
Features

Returns previous PSYCHLOPS problem description to the user
PSS-10 + PSYCHLOPS scoring
App engagement tracking
Personalized progress summary on completion

Stack

HTML/CSS/JS → GitHub → Netlify
Netlify Functions for Airtable read/write
Airtable base: appz0D6kUon2e70B5

Part of the Places to Thrive / UnBroken platform
